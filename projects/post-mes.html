BOM은 자재명세서로 제품의 생성부터 폐기까지 제품정보와 속성을 지속적으로 관리합니다. E-BOM(설계), S-BOM(영업)을 맡아 개발을 진행했고, BOM 버전관리와 견적서 생성이 주요 기능이었습니다.

메일은 Linux SendMail(SMTP)과 Dovecot(IMAP)을 이용했습니다.

Technologies:
  • - Spring Boot Framework
  • - Jqx Grid
  • - SendMail, Dovecot
