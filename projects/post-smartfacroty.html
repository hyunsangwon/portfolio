IoT 스마트 팩토리 데모

IoT 스마트 팩토리는 2021년 5월부터 8월까지 약 2개월간 진행했고, 사업형 국가과제 프로젝트입니다. 총 2명(펌웨어 개발자 포함)이서 개발을 진행했고 그중, BLE Gateway, 백엔드(AWS 포함), 프론트엔드를 맡아 개발을 지휘 했습니다.

팩토리 특성상 네트워크 대역폭이 제한되는 곳에 있기 때문에 사내에서 처음으로 RAW패킷이 아닌 MQTT 프로토콜을 선택했고, 디바이스 전력을 절약하기 위해 BLE 통신을 선택했습니다.

Gateway는 Raspberry를 사용했고, BLE는 Python에 bleak 라이브러리를 사용했습니다. 디바이스에서 받은 Hex데이터를 파싱하여 MQTT로 AWS IoTCore에 전송했습니다.

8월 1차데모와 12월 2차데모 예정 이었으나, 1차 데모를 끝으로 회사를 그만두게 되었습니다.

Technologies:
  • - TypeScript
  • - Nest.js
  • - Python
  • - BLE Gateway(Raspberry pi 4)
  • - MQTT
  • - MySQL
  • - TypeORM
  • - AWS (Lambda & DynamoDB)
  • - AWS (IoTCore)
  • - AWS (EC2)
  • - KAKAO Maps API
  • - OS(Ubuntu)
