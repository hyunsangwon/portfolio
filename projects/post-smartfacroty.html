IoT 스마트 팩토리

IoT 스마트 팩토리는 2020년 6월부터 7월까지 약 2개월간 진행되었고, 삼성프린터 납품 업체를 대상으로 만든 프로젝트 입니다. 처음으로 PL을 맡아 진행했고 개발 인원은 총 3명으로 진행했습니다.

차량에 IoT디바이스를 부착하면 IoT디바이스가 RAW데이터를 AWS Server에 보내고 AWS Lambda에서 RAW데이터를 파싱및 필터링하여 DB에 데이터를 저장합니다.

DB에 쌓인 데이터는 배치서버에서 주기적으로 2차 필터링을 하여 최종 웹에 데이터를 제공합니다.

사용자는 운행 이력과 현재 위치, 총 이동 거리를 검색할 수 있고, 지역 설정 기능이 있어 경로 이탈한 차량을 찾아낼 수 있습니다.

Technologies:
  • - TypeScript
  • - Nest.js
  • - Python
  • - BLE Gateway(Raspberry pi 4)
  • - MQTT
  • - MySQL
  • - TypeORM
  • - AWS (Lambda & DynamoDB)
  • - AWS (IoTCore)
  • - AWS (EC2)
  • - KAKAO Maps API
  • - OS(Ubuntu)
