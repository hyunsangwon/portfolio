차량관제 프로젝트는 차량을 관리하는 업체를 대상으로 만든 프로젝트 입니다.

데이터 플로우는 디바이스를 차량 USB에 연결 후 RAW 데이터를 소켓 서버로 전송합니다. 서버에서 받은 데이터는 AWS DynamoDB에 저장되고 트리거 이벤트로 AWS Lambda에 연결하여 데이터 필터링을 거쳐 다시 AWS DynamoDB에 저장되는 프로세스 입니다.

카카오맵 API을 이용하여 차량 위치, 이동 경로, 이동 거리를 기능을 구현했습니다.

차량관제 프로젝트는 디바이스 배터리 이슈와 크기 문제로 임시 중단 되었습니다. 디바이스에 고정 IP를 포팅하는데, 고정 IP를 새로 발급할 때 마다 매번 포팅하는 작업이 불편했고 도메인으로 포팅했으면 테스트할 때 더 편했을 것 같다는 생각이 들어 아쉬웠습니다.

프로젝트를 진행하며 소켓 데이터 파싱하는 방법과 애자일 개발에 필요성을 알게 되었습니다. 프로젝트 샘플은 Click WEB site 에서 확인할 수 있습니다.

Technologies:
  • - Spring Boot Framework
  • - AWS (Lambda & DynamoDB)
  • - AWS (EC2 & S3 & RDS)
  • - KAKAO Maps API
  • - OS (CentOS7)
