차량관제 시스템

차량관제 프로젝트는 2020년 6월부터 7월까지 약 2개월간 진행되었고, 삼성프린터 납품 업체를 대상으로 만든 프로젝트 입니다. 처음으로 PL을 맡아 진행했고 개발 인원은 총 3명으로 진행했습니다.

차량에 IoT디바이스를 부착하면 IoT디바이스가 RAW데이터를 AWS Server에 보내고 AWS Lambda에서 RAW데이터를 파싱및 필터링하여 DB에 데이터를 저장합니다.

DB에 쌓인 데이터는 배치서버에서 주기적으로 2차 필터링을 하여 최종 웹에 데이터를 제공합니다.

사용자는 운행 이력과 현재 위치, 총 이동 거리를 검색할 수 있고, 지역 설정 기능이 있어 경로 이탈한 차량을 찾아낼 수 있습니다.

사용자가 디바이스의 크기와 전원 USB케이블의 불편함을 제기하여 개발 도중 중단 되었습니다.

프로젝트 기간동안 RAW데이터 파싱하는 방법과 애자일 개발에 필요성을 알게 되었습니다. 프로젝트는 Click WEB site 에서 확인할 수 있습니다.

Technologies:
  • - JAVA
  • - Spring Boot Framework
  • - Android with Kotlin
  • - Thymeleaf
  • - Postgresql
  • - MyBatis
  • - AWS (Lambda & DynamoDB)
  • - AWS (EC2 & S3 & RDS)
  • - KAKAO Maps API
  • - OS(CentOS7)
