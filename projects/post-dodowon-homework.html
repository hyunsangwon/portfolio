사내 결재 시스템

팀 프로젝트 참여하기 전에 개인 평가를 하기위해 만든 프로젝트 입니다. 시간은 한달을 주셨고, 언어는 Java와 Spring Boot + MyBatis + Maven을 사용해야 했습니다.

시간이 한 달이었기에, 프론트는 Bootstrap의 무료 템플릿을 사용해 커스텀 했습니다. 회사에서 사용하는 MVC 패턴을 벗어나지 않게 코드를 짜며, 개인 사이드 프로젝트를 하면서 추가해보고 싶었던 Google reCAPTCHA, GraphQL, EhCache, yaml를 프로젝트에 시도해보았습니다. Spring Security에서 특히 예상치 못하게 시간이 걸려 어려움을 겪었는데, 주말까지 공부하며 제가 느낀 경험을 블로그에 포스팅 한게 기억에 가장 남습니다. Click Site Spring Boot로 개발하면 Spring 레거시에서 XML에서 설정했던 프로퍼티를 Class로 구현하기에 디버깅이 가능하고 내장 톰캣이 있어 톰캣 설정없이 실행 및 jar 배포는 정말 큰 매력이였습니다.

프로젝트 소스는 Click GitHub 에서 확인할 수 있습니다.

Technologies:
  • - JAVA
  • - Spring Boot Framework
  • - Thymeleaf
  • - MySQL
  • - MyBatis
  • - GraphQL
