QR 재고 프로젝트는 저가나 이동성이 낮은 자산, 원재료·반제품·완제품 등 재고자산에 QR스티커를 부착하여 관리하는 사업형 국가과제 프로젝트입니다.

사용자는 주기적으로 QR을 스캔하여 자산위치를 신고합니다. QR에 자산 정보와 위,경도 값을 인코딩하여 관리자는 자산위치와 수량을 모니터링할 수 있습니다.

해당 프로젝트는 정량적 정성적 목표치 통과가 목표였기에 사용하고 싶었던 기술을 도입했습니다. 요청이 많은 API를 AWS Lambda를 이용해 처리한게 가장 기억에 남고 Serverless Architecture에 장·단점을 파악하게 되었습니다.

Technologies:
  • - Spring Boot Framework
  • - Android with Kotlin
  • - MySQL
  • - AWS (Rekognition)
  • - AWS (EC2,S3,RDS)
  • - AWS (Lambda,API Gateway)
  • - KAKAO Maps API
  • - OS (CentOS7)
