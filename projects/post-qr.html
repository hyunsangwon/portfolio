QR 재고관리 시스템

QR 재고 프로젝트는 2020년 1월부터 7월까지 약 7개월간 진행했고, 삼성 프린터 납품 업체를 대상으로한 사업형 국가과제 프로젝트 입니다. 개발 인원은 총 4명이고, API Backend와 AWS를 맡아 업무를 진행 했습니다.

QR에는 최대 2953 바이트를 인코드할 수 있었기 때문에 자산 출처를 입력하는데 제한이 있어 특정 시리얼 번호를 인코드 하여 대량으로 스티커를 출력했습니다.

QR스티커를 붙인 재고자산은 앱을 이용하여 등록및 관리하며, 재고정보는 AWS rekognition OCR 기능을 사용하여 저장할 수 있게 구현했습니다.

사용자는 재고 수량, 자산 관리자, 자산 위치, 자산 종류를 웹에서 확인및 모니터링 가능합니다.

삼성 프린터 업체에서 프린터 잉크잔량 모니터링하는 프로세스까지 원해 사업은 무산되었지만, 과제는 통과했습니다. 해당 프로젝트로 정량적 정성적 목표, AWS Serverless Architecture를 알게 되었습니다.

Technologies:
  • - JAVA
  • - Spring Boot Framework
  • - Android with Kotlin
  • - Thymeleaf
  • - MySQL
  • - MyBatis
  • - AWS (Lambda & DynamoDB & Rekognition)
  • - AWS (S3 & RDS)
  • - AWS (EC2 & API Gateway)
  • - KAKAO Maps API
  • - OS(CentOS7)
